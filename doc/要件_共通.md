# 1. 共通機能

- エラーハンドリング
  - Notificationコンポーネントで表示できるものは表示
  - それ以外はエラー画面へ
- input
  - バリデーションエラー
    - 基本ブラウザのデフォルトエラー表示でOK
    - 何か込み入ったことをする場合は赤文字でエラーメッセージを表示（なるべくMantineのInputのエラーメッセージ表示で表示する）
- ページネーション
  - 30件ずつ表示
- フィルター
  - indexを貼った一部のカラムのみ対応
- ソート
  - indexを貼った一部のカラムのみ対応
